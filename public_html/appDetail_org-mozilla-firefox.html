Firefox ブラウザの公式ページへようこそ。
Android 版 Mozilla Firefox は、デスクトップ版の優れた特長を Android へ持ち込みました。高速で使いやすく、カスタマイズ自由自在。Web を安心して利用できるよう、最新のセキュリティ・プライバシー機能を備えています。
インストール前に必ず対応機種 http://mozilla.jp/firefox/android/devices/ をご確認ください。

また、何かフィードバックがありましたら http://mzl.la/FXinput からお寄せください。皆さんのブラウジング体験をさらに改善するための参考とさせていただきます。

【Firefox ブラウザの主な機能と特長】

・高速
Web 上で目的の情報へすばやく辿り着けます。

・同期
デスクトップ版 Firefox のブックマーク、履歴、タブ、パスワードをどの端末でも共有できます。

・高度なセキュリティ

プライバシーやセキュリティ、Web 上での個人情報の扱いなどを管理できます。

・スマートスクリーン
最小限の文字入力で、あるいは一切入力をせずに、お気に入りのサイトへアクセスできます。

・アドオン
ブラウザを自分好みにカスタマイズできる拡張機能が揃っています。

詳しい機能紹介は http://mozilla.jp/firefox/android/features/ をご覧ください。

【サポート】

質問があるときは http://mozilla.jp/firefox/support/ で検索してみてください。

対応機種の一覧は http://mozilla.jp/firefox/android/devices/ に掲載しています。

Firefox が要求する許可設定の詳細は https://support.mozilla.org/ja/kb/how-firefox-android-use-permissions-it-requests をご覧ください。

【Mozilla について】

Mozilla は、Web の力を人々の手にとどめることに専念している非営利法人であり、すべての人のために革新を続けている、ユーザ、貢献者、開発者を中心としたグローバルなコミュニティでもあります。Firefox を使っていただくことで、あなたもそのコミュニティの一員となり、それが Web の明るい未来を築くための支援にもつながります。詳しくは http://mozilla.jp/about/ をご覧ください。
